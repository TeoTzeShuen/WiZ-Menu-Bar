# MacOS Menu Bar WiZ Light Controller
Small menu bar application that broadcasts UDP commands to WiZ-Enabled Lightbulbs

UDP Commands were taken from https://seanmcnally.net/wiz-config.html

**Includes features such as:**
 - Lightbulb control (!)
 - Auto discovery of WiZ bulbs on local network
 - RGBWW Settings
 - Small widget to monitor lightbulb statuses
